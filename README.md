# Анализ влияния наличия сайта на популярность юридических фирм

**Pet-project** для портфолио аналитика данных

**Цель проекта**

Понять, влияет ли наличие собственного сайта у юридической фирмы на её популярность и клиентскую активность.
В качестве прокси-метрики популярности использовалось количество отзывов на Яндекс.Картах.

**Ключевые выводы**
- Фирмы с сайтом имеют в среднем **в 2 раза больше отзывов** (42.6 против 21.3)
- Медианное количество отзывов отличается в **8 раз** (8 против 1)
- Доля компаний с 5+ отзывами: **61%** (с сайтом) против **38%** (без сайта)
- Доля фирм без единого отзыва: **26%** (с сайтом) против **47.5%** (без сайта)
- Различия статистически значимы: Mann-Whitney U p-value = **2.17 × 10⁻⁵⁷**
→ Наличие сайта сильно коррелирует с большей видимостью и клиентской активностью.

**Данные**
~5359 организаций (Москва и область), собранные с Яндекс.Карты в 2026 году.
После очистки и удаления дублей — **4421** записей для анализа.

**Что было сделано**
1. **Сбор данных** — многопоточный парсер Яндекс.Карты
   Репозиторий парсера: https://github.com/AlpakaGSB/parser_2
2. **Очистка и подготовка** — PostgreSQL + SQL (приведение типов, флаги, удаление дублей)
3. **Исследовательский анализ и визуализация** — Python (pandas, seaborn, matplotlib)
4. **Статистическое тестирование** — Mann-Whitney U, z-тест долей, bootstrap CI и др.
5. **Выводы и рекомендации** — оформлены в ноутбуке и README

**Технологии**
- Python 3 (pandas, seaborn, matplotlib, scipy, sqlalchemy)
- PostgreSQL + SQL (очистка, витрины)
- Jupyter Notebook
- Git / GitHub
- Многопоточный парсер (requests, Selenium, BeautifulSoup)

**Ограничения исследования**

- Корреляция ≠ причинно-следственная связь
- Данные только из Яндекс.Карты (возможен bias)
- Отзывы — косвенная метрика посещаемости и популярности
- Парсинг выполнен исключительно в учебных целях
